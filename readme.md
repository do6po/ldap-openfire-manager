REQUIREMENTS
------------

php >= 7.1.0


INSTALLATION
------------

### Install via git

```bash
git git clone git clone https://helvis@bitbucket.org/helvis/ldap-openfire-manager.git
cd ldap-openfire-manager
```

### Install vagrant plugins

```bash
vagrant plugin install vagrant-hostmanager
vagrant up
```

### Start vagrant

```bash
vagrant up
```
